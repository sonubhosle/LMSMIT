# Library Management System

A complete desktop library management application built with Electron, Node.js, Express, MongoDB, and Tailwind CSS.

## Features

- **Authentication System**: Secure librarian login and registration
- **Student Management**: Add, edit, delete, and search students
- **Book Management**: Complete book inventory with quantity tracking
- **Book Issuance**: Issue books to students with due dates
- **Penalty System**: Automatic penalty calculation for late returns
- **Invoice Generation**: Automatic invoices for late returns and lost books
- **Excel Export**: Export data to Excel for all modules
- **Print Functionality**: Print invoices and lists
- **Real-time Updates**: Auto-refresh UI without page reload
- **Modern UI**: Beautiful interface with Tailwind CSS and animations
